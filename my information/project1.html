Car Dealership Website

A simple static website created using only HTML & CSS

📌 Project Overview

This project is a static website designed for a fictional car dealership company. It showcases different car models, their features, a gallery, contact form, and a company overview — all designed using only HTML and CSS.

🛠️ Tools & Technologies Used

  • HTML5: Used for structuring the content of the website
  • CSS3: Used for styling and layout, including colors, spacing, and responsiveness

💡 Key Features

  • Homepage with banner and navigation
  • Car models listed with details
  • Responsive design using CSS Flex/Grid
  • About and Contact sections
