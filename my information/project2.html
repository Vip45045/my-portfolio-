Mumbai Dabbawala Website

Built using HTML, Bootstrap 4.1.3 & Animate.css

📌 Project Overview

This project is a tribute website for Mumbai’s famous Dabbawalas. It highlights their history, operations, and efficiency using visually appealing sections with animation. The entire website is made responsive using Bootstrap 4, styled with custom CSS, and enhanced with Animate.css for engaging effects.

🛠️ Tools & Technologies Used

  • HTML5: Structure and layout
  • CSS3: Custom styles
  • Bootstrap 4.1.3: Responsive design and layout utilities
  • Animate.css: Animations for entrance effects, transitions, etc.

📷 Screenshot Preview

💡 Key Features

  • Animated header and sections
  • Fully responsive layout for all devices
  • Social media icon section with hover effects
  • Informative content about Dabbawala culture
  • Bootstrap cards and sections for clean structure
